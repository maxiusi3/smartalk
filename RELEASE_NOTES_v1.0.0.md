## 📋 升级指南

### 🔄 从旧版本升级
如果您正在使用SmarTalk的早期版本，请按照以下步骤升级：

1. **备份数据**：备份您的学习进度和个人设置
2. **更新代码**：`git pull origin main`
3. **安装依赖**：`npm install`
4. **数据迁移**：运行 `npm run migrate`
